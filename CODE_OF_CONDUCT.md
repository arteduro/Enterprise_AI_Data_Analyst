# Código de Conducta

## Nuestro compromiso

Nos comprometemos a mantener un entorno abierto, inclusivo y respetuoso para todos los colaboradores.

## Comportamiento esperado

- Ser respetuoso con todos.
- Aceptar críticas constructivas.
- Compartir conocimiento.
- Ayudar a mejorar el proyecto.

## Comportamiento no permitido

- Acoso.
- Discriminación.
- Lenguaje ofensivo.
- Ataques personales.

## Aplicación

Los mantenedores podrán eliminar contribuciones que incumplan este código.
